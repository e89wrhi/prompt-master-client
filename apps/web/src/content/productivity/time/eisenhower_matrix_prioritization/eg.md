# Examples: Eisenhower Matrix Prioritization

## 1. Marketing Manager Inbox Overload

```text
Act as a Senior Systems Thinker. Prioritize the following tasks:

### The Backlog:
- Responding to client feedback for Tuesday launch.
- Reviewing 2024 content strategy.
- Replying to 10 "Quick Question" emails from sales team.
- Updating my LinkedIn banner.
- Finding a new CRM for next year.
- Booking a flight for a conference in 4 months.

### Prioritization Requirements:
1. **Q1 (DoFirst)**: Launch-critical feedback.
2. **Q2 (Schedule)**: 2024 strategy, CRM research.
3. **Q3 (Delegate)**: Sales emails (Give them a FAQ?).
4. **Q4 (Eliminate)**: LinkedIn banner update (Is it really moving the needle?).
5. **The Strategic Pivot**: How to move "CRM research" into a recurring Q2 slot.

### Output Constraints:
- 2x2 Markdown Matrix format.
- Unsentimental tone.
```

## 2. Solopreneur Project Backlog

```text
Act as a Management Consultant. Prioritize my project backlog:

### The Backlog:
- Fixing a broken checkout button (Site losing money).
- Writing next week's newsletter.
- Redesigning the website footer.
- Pitching 3 new enterprise clients.
- Scrolling Twitter for "inspiration."

### Prioritization Requirements:
- Use the Eisenhower Matrix categories.
- Suggest "The Action Plan": Top 3 immediate tasks.

### Output Constraints:
- Categorized list format.
```
