# Examples: Pomodoro Schedule Creation

## 1. Deep Work: Writing a Technical Whitepaper

```text
Act as a Focus Specialist. Create a structured Pomodoro Schedule for a 3-hour deep work session.

### Goal:
Write 1,500 words for the "Future of Edge Computing" whitepaper.

### Tasks:
- Outline: 20 mins.
- Section 1 (Current State): 60 mins.
- Section 2 (Challenges): 60 mins.
- Proofreading: 20 mins.

### Schedule Requirements:
1. **Interval Strategy**: 50/10 split (Needs deep flow).
2. **Restorative Breaks**: Stretching, walking to get water, deep breathing. No phone.
3. **The "Flow State" On-Ramp**: Clearing desk and putting on brown noise.
5. **The Mid-Point Review**: Check word count after block 2.

### Output Constraints:
- Block-by-block format.
- Calm and High-Focus tone.
```

## 2. Admin Batching: Clearing a Messy Inbox

```text
Act as a Focus Specialist. Create a Pomodoro Schedule for a 1-hour "Admin Sprint."

### Tasks:
- Reply to 15 pending emails.
- Update project timeline in Jira.
- Schedule 3 meetings.
- File expense report.

### Schedule Requirements:
1. **Interval Strategy**: 25/5 split (Fast-paced tasks).
2. **Batching Protocol**: Group all email replies into the first 2 Pomodoros.
3. **Restorative Breaks**: Look out the window for 5 mins (Eye rest).

### Output Constraints:
- Block-by-block format.
```
