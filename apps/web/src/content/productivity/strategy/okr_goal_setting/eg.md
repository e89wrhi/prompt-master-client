# Examples: OKR Goal Setting

## 1. Engineering Team (Focus on Reliability)

```text
Act as a Chief Operating Officer (COO). Facilitate a structured OKR Setting session for the Engineering Team for Q2 2024.

### Current Vision & Context:
- Company Goal: Scale to 1M users.
- Main Challenge: Recent performance degradation and "spaghetti code" slowing down features.

### OKR Requirements:
1. **The Core Objective (O)**: Create an objective around "World-Class Infrastructure Reliability."
2. **The Key Results (KRs)**:
   - Increase uptime from 98.5% to 99.95%.
   - Reduce average API latency from 400ms to <150ms.
   - Achieve 100% test coverage for all Tier-1 payment services.
3. **Strategic Initiatives**: Implementing Kubernetes auto-scaling and a "Testing Tuesday" culture.
4. **Alignment Check**: How this supports the mission of "Frictionless Global Payments."

### Output Constraints:
- Use a Rigorous and Structured tone.
- Format the final OKRs in a table.
```

## 2. Content Marketing Team (Focus on Organic Growth)

```text
Act as a Performance Coach. Facilitate a structured OKR Setting session for the Content Team for Q1.

### Current Vision & Context:
- Goal: Reduce reliance on paid ads.
- Challenge: High bounce rate on the company blog.

### OKR Requirements:
1. **The Core Objective (O)**: Objective: "Become the #1 Educational Authority in the [INDUSTRY] Niche."
2. **The Key Results (KRs)**:
   - Increase monthly organic traffic from 10k to 25k.
   - Earn 50 high-quality backlinks from Domain Authority (DA) 60+ sites.
   - Increase newsletter sign-up conversion rate from 2% to 5%.
4. **Alignment Check**: Supports the mission of "Empowering users with knowledge."

### Output Constraints:
- Inspirational and Metric-led tone.
```
