# 📊 Analysis Tools

Prompts for quantitative and qualitative data digestion, root cause isolation, and financial reasoning.

## Available Categories

- [**Cost Benefit Analysis**](./cost_benefit_analysis/index.md): Listing hard/soft costs and projected ROI.
- [**CSV Data Interpretation**](./csv_data_interpretation/index.md): Dropping raw CSV data and asking for trend insights.
- [**Root Cause Analysis (5 Whys)**](./root_cause_analysis_5whys/index.md): Forcing the AI to dig past symptoms to systemic issues.
- [**SWOT Matrix Generation**](./swot_matrix_generation/index.md): Automatically identifying Strengths, Weaknesses, Opportunities, and Threats.
