# 📑 Summarization Tools

Providing structure to AI for digesting incredibly long transcripts, textbooks, and videos.

## Available Categories

- [**Key Takeaways Extraction**](./key_takeaways_extraction/index.md): Bullet-point synthesis without context loss.
- [**Lecture Transcript Digests**](./lecture_transcript_digests/index.md): Turning raw whispered/auto-generated transcripts into standard notes.
- [**Textbook Chapter Summaries**](./textbook_chapter_summaries/index.md): Analyzing structure and bolded vocabulary in chapters.
- [**Youtube Video Notes**](./youtube_video_notes/index.md): Summarizing standard educational video transcripts.
