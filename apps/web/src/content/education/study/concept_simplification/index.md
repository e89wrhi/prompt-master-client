# Template: The Feynman Technique (Concept Simplification)

Use this template to break down complex, "impenetrable" topics into simple, clear explanations. It is designed to utilize the "Feynman Technique"—stripping away jargon, using relatable analogies, and explaining a concept with such clarity that even a non-expert (or a second-grader) could grasp the core logic.

## 📋 Prompt Template

```text
Act as a Master Educator and expert in the Feynman Technique. Simplify the following [COMPLEX_TOPIC].

### Instructional Goal:
Provide a three-tiered explanation for a [TARGET_AUDIENCE]:
1. **The "Second-Grader" Summary (ELI5)**: Use a physical, real-world [ANALOGY] to explain the core principle without using any technical jargon.
2. **The "High-School" Bridge**: Introduce 2-3 [CORE_TECHNICAL_TERMS] and explain them using simple language.
3. **The "Final Synthesis"**: Provide a cohesive, 1-paragraph explanation that is technically accurate but remains readable.

### Simplification Requirements:
- **No Jargon**: If you must use a complex word, you MUST define it immediately.
- **Relatability**: Compare the topic to something common (e.g., mail delivery, a library, baking a cake).
- **The "Struggle" Spot**: Identify the most confusing part of [COMPLEX_TOPIC] and focus the heaviest simplification effort there.

### Output Constraints:
- Use a Friendly, Encouraging, and Crystal-Clear tone.
- Conclude with a "Knowledge Check"—a single question the user should be able to answer after reading this.
```

## 🧩 Variables to Fill Out

- `[COMPLEX_TOPIC]` - e.g., Quantum Entanglement, The 2008 Financial Crisis, Recursion in Coding.
- `[TARGET_AUDIENCE]` - Who is learning this?
- `[ANALOGY]` - e.g., A post office, a game of telephone, a mirror.
- `[CORE_TECHNICAL_TERMS]` - e.g., Entanglement, Sub-prime, Base case.

## 💡 Pro-Tips

- **Analogies are "Fragile":** An analogy is never perfect. Ask the AI to "Identify one way the analogy fails" so the student doesn't develop a misconception.
- **Visual Description:** Even without an image, ask the AI to "Describe what a child would draw on a whiteboard to represent this idea."
- **The "Why" Matters:** People remember logic better than facts. Ask the AI to "Explain the 'Why' behind the concept (e.g., 'Why do we need recursion?')."
