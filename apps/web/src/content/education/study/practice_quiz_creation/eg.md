# Examples: Adaptive Practice Quizzes

## 1. AWS Certified Cloud Practitioner

```text
Act as a Test Developer. Generate a 10-question practice quiz for the "Cloud Concepts" section of the AWS CCP exam.

### Quiz Context:
- **Difficulty**: Professional Certification Level.
- **Format**: Multiple Choice (4 options).

### Question Architecture:
1. **Stem**: Clear questions about S3, EC2, and the Shared Responsibility Model.
2. **Distractors**: Highly plausible (e.g., confusing IaaS with PaaS).
3. **Rationale**: Included for every answer.
4. **Hint**: Provided for the hardest 3 questions.

### Output Constraints:
- Precise and Evaluative tone.
- Answer Key at the bottom.
```

## 2. Advanced Psychology (Cognitive Biases)

```text
Act as an Educational Stylist. Generate a quiz on "The Confirmation Bias and Sunk Cost Fallacy."

### Quiz Context:
- **Format**: Scenario-based.

### Output Constraints:

```
