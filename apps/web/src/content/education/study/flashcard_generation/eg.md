# Examples: High-Retention Flashcard Generation

## 1. Medical Terminology (Anki Style)

```text
Act as a Memory Expert. Generate 10 flashcards for "Anatomy 101: The Nervous System."

### Material Scope:
- Topics: Neurons, synapses, myelin sheath, neurotransmitters.

### Flashcard Anatomy:
- Front: Cloze deletion style ({{...}}).
- Back: Concise (3-5 words).
- Mnemonic: Included for the tricky terms.

### Technical Constraints:
- Markdown Table format.
- "Atomic" cards only.

### Output Constraints:
- Clinical tone.
```

## 2. Spanish Vocabulary (Beginner)

```text
Act as a Learning Scientist. Generate 20 Spanish vocabulary cards.

### Material Scope:
- Common household items (Table, Chair, Kitchen, etc.).

### Flashcard Anatomy:
- Front: Spanish word.
- Back: English translation + small example sentence.

### Technical Constraints:
- CSV format for Anki import.
```
