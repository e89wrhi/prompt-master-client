# 🧠 Study Aids & Retention

Prompt templates designed to help students memorize, drill, and understand complex content efficiently.

## Available Categories

- [**Concept Simplification**](./concept_simplification/index.md): The "Explain Like I'm 5" (ELI5) frameworks for tough topics (Feynman Technique).
- [**Flashcard Generation**](./flashcard_generation/index.md): Converting notes into Anki-compatible CSVs.
- [**Practice Quiz Creation**](./practice_quiz_creation/index.md): Generating multiple-choice questions with realistic distractors.
- [**Spaced Repetition Schedules**](./spaced_repetition_schedules/index.md): Building optimal review calendars for exams.
