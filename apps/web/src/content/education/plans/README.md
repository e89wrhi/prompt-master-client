# 📝 Lesson Plans & Educator Tools

Prompts aimed at teachers and professors to help offload the administrative burden of curriculum design.

## Available Categories

- [**Curriculum Development**](./curriculum_development/index.md): Structuring semester-long or year-long educational goals.
- [**Grading Rubrics**](./grading_rubrics/index.md): Generating fair, explicit rubrics for essays and projects.
- [**Lesson Plan Generation**](./lesson_plan_generation/index.md): Creating period-specific activities aligned with core standards.
- [**Syllabus Structuring**](./syllabus_structuring/index.md): Generating high-level course overviews and policies.
