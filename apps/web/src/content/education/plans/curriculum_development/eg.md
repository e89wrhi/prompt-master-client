# Examples: Strategic Curriculum Development

## 1. 12-Week Full-Stack Web Dev Bootcamp

```text
Act as a Senior Curriculum Architect. Develop a 12-week Curriculum for "Modern Full-Stack Engineering."

### Course Context:
- **Target Learner**: Career switchers with zero coding background.
- **Goal**: Taking a student from "Hello World" to a deployed SaaS app.
- **Final Outcome**: A full-stack E-commerce platform with a working database and payment gateway.

### Curriculum Requirements:
1. **Modules**: 6 modules (2 weeks each).
   - Module 1: The UI (HTML/CSS/Tailwind).
   - Module 2: The Logic (Advanced JS).
   - Module 3: The Framework (React).
   - Module 6: The Capstone (Deployment).
2. **Assessment**: Bi-weekly coding sprints.
3. **Prerequisite**: Basic computer literacy.

### Output Constraints:
- Detailed Table format.
- Academic and Structured tone.
```

## 2. 4-Week "Intro to Stoic Philosophy" (Adult Learning)

```text
Act as a Dean. Develop a 4-week Curriculum for "Applied Stoicism for Modern Leaders."

### Course Context:
- **Audience**: High-stress executives.
- **Goal**: Improving emotional resilience and decision-making.

### Curriculum Requirements:
- 4 Modules (Weekly).
- Final Outcome: A "Personal Crisis Handbook" written by the student.
- Assessment: Weekly journaling exercises.

### Output Constraints:
- Professional tone.
```
