# Examples: Professional Grading Rubrics

## 1. Technical Coding Project (React Web App)

```text
Act as a Lead Educator. Develop a Grading Rubric for the Coding Project: "Personal Portfolio Site."

### Assignment Context:
- **Goal**: Demonstrate mastery of React components and responsive CSS.
- **Level**: Junior Developer Bootcamp.

### Rubric Requirements:
1. **Criteria**: 
   - Component Architecture (30%).
   - Responsive Styling (30%).
   - Code Documentation (20%).
   - UX/UI Polish (20%).
2. **Levels**: Excellent (A) to Beginning (D).
3. **Common Pitfalls**: No responsive design, broken image links, hard-coded data.
4. **Feedback Starters**: Included.

### Output Constraints:
- Visual Markdown Table.
- Neutral and Standardized tone.
```

## 2. Analytical Research Essay

```text
Act as a Quality Assurance Specialist. Develop a rubric for a 2,500-word Research Essay on "The Ethics of AI."

### Assignment Context:
- **Level**: Graduate (Master's).
- **Goal**: High-level critical synthesis of existing literature.

### Rubric Requirements:
- Criteria: Literature Review Depth, Logical Argumentation, Citation Accuracy (APA 7), Original Contribution.
- Focus on "Rigorous evidence-based claims" for the A level.

### Output Constraints:
- Precise and Standardized tone.
```
