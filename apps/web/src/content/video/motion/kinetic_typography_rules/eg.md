# Kinetic Typography Examples

## 1. Aggressive High-Retention Pop-Up Subtitles
**Prompt:**
> "Outline the keyframe animation rules to generate aggressive TikTok style pop-up subtitles.
> - **Typography & Layout:** Font family: Montserrat Black. Tracking: -20. Anchor Point: Bottom Center.
> - **In-Animation:** Animate the Scale property from 70% to 110%, then settle back to 100%. Use an Animator property targeting 'By Word'.
> - **Easing:** Apply a spring-bounce easing curve (fast entry, slight overshoot, quick settle back).
> - **Motion Polish:** Apply a drop shadow (Opacity 100%, Distance 5, Size 0) for maximum contrast against busy B-roll."

## 2. Elegant Cinematic Reveal
**Prompt:**
> "Outline the keyframe animation rules to generate elegant cinematic titles.
> - **Typography & Layout:** Font family: Cinzel Decorative. Tracking: +300. Anchor Point: Center.
> - **In-Animation:** Animate the Opacity from 0 to 100 and Blur from 20 to 0. Use an Animator property targeting 'By Character'. Add an Offset keyframe passing from -100% to 100% over 3 seconds.
> - **Easing:** Use perfectly linear interpolation for a constant, graceful reveal.
> - **Out-Animation:** Fade Opacity to 0 over 2 seconds globally (no character offset)."
