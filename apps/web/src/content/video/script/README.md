# 📝 Video Scripts & Screenwriting: Detailed Guide

Writing for video is not just about words—it's about the relationship between the ear and the eye. This guide focuses on Audio/Visual (AV) synchronization.

---

## 📈 Level-Based Scripting Strategy

### 🐣 Level 1: Content Draft
**Focus:** Information accuracy.
**Example Prompt:** *"Write a 500-word script about the history of coffee according to the provided notes."*

### 👟 Level 2: Conversational VO
**Focus:** Natural delivery.
**Example Prompt:** *"Rewrite this script for a YouTube video. Use contractions, shorter sentences, and a friendly, upbeat tone."*

### 🏢 Level 3: Visual Mapping
**Focus:** B-Roll and Callouts.
**Example Prompt:** *"Take this script and add B-roll callouts for every 5 seconds. Use 'Archival' footage style for the history segments."*

### 🏛️ Level 4: High-Retention Framework
**Focus:** Strategy and retention arcs.
**Example Prompt:** *"Construct an AV script for a tech documentary. Use a 15-second high-stakes hook, force a pattern interrupt every 45 seconds, and map visuals to exactly 2 sentences of VO per row in a table."*

---

## 🛠️ Advanced Techniques

| Technique | Description | AI Benefit |
| :--- | :--- | :--- |
| **AV Table Format** | Force a two-column Markdown table (Visuals vs. Audio). | Perfect alignment between words and shots. |
| **Pause Directives** | Use `[BEAT]` or `[LONG PAUSE]` in VO text. | Natural pacing instead of a relentless wall of sound. |
| **Agitate-Solve** | Open with the problem before the solution. | Higher viewer retention in explainer and sales formats. |
