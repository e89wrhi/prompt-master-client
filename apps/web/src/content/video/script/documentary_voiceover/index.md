# Template: Documentary Voiceover

Use this template to generate dramatic, compelling narration scripts for long-form documentary content, focusing strongly on cadence, thematic weight, and emotional arcs.

## 📋 Prompt Template

```text
Act as a Senior Documentary Screenwriter. Write an opening voiceover narration detailing [CORE_THEME].

### Script Requirements:
1. **Cadence & Tone**: The narration must feel [TONE]. Use [SENTENCE_STRUCTURE] sentence structures to maintain audience engagement and gravitas.
2. **Perspective**: Write from the perspective of an [POV] narrator.
3. **Thematic Hook**: Start explicitly with a [HOOK_TYPE] to instantly grip the viewer.
4. **Directives**: Include subtle `[BEAT]` or `[LONG PAUSE]` markers where the narrator should let the visuals breathe or allow the music to swell.

### Output Constraints:
Do not use conversational web-friendly filler (e.g., "Hey guys," "So in this video"). Ground the dialogue in cinematic prestige. Output approximately [WORD_COUNT] words.
```

## 🧩 Variables to Fill Out

- `[CORE_THEME]` - Main subject (e.g., The isolation of deep-sea diving, The rise of AI, The Apollo 11 silence).
- `[TONE]` - Emotional feeling (e.g., haunting, triumphant, contemplative).
- `[SENTENCE_STRUCTURE]` - Writing style (e.g., short punchy, flowing poetic, journalistic).
- `[POV]` - Point of view (e.g., omniscient, first-person survivor, objective historian).
- `[HOOK_TYPE]` - How to open (e.g., cold hard statistic, philosophical question, sensory description).
- `[WORD_COUNT]` - Desired length (e.g., 200 words, 500 words).

## 💡 Pro-Tips
- **Pause Markers:** AI tends to write a wall of text. Forcing it to insert `[BEAT]` or `[SOUND DESIGN]` markers prevents the voiceover from feeling relentlessly rushed.
- **Rhythm matters:** Combining long, poetic sentences with brief, punchy statements creates a dynamic rhythm that voice actors love.
