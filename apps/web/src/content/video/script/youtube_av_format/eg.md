# Examples: YouTube A/V Format

## 1. High-Retention Tech Essay
```text
Act as a Top 1% YouTube Strategist and Scriptwriter. Generate an A/V format script for a video about Why 99% of Consumer Drones Are Useless.

### Format Guidelines:
Generate a markdown table with two columns: **Visuals** (Left) and **Audio/VO** (Right).

### Script Structure:
1. **The Hook (0-15s)**: In the Audio column, immediately introduce the illusion of drone tech marketing. In the Visual column, specify a fast-paced A-roll setup with a drone crashing into a tree.
2. **The Context (15-60s)**: Provide the background information using a conversational, storytelling cadence. Describe the exact on-screen slick 3D tracking graphics needed to support the VO.
3. **The Transition**: Use a pattern interrupt: a sudden record scratch and dead silence before moving into the main content block.
4. **Pacing Rules**: Do not let any single Audio row exceed 2 sentences before forcing a new visual directive in the left column.

### Output Constraints:
Produce a comprehensive 3-minute script framework. Ensure the Audio sounds raw and conversational, tailored to a dry witty tech-essayist persona.
```
