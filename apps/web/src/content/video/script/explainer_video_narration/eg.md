# Examples: Explainer Video Narration

## 1. Zero-Trust Security Explainer
```text
Act as a B2B SaaS Copywriter and Explainer Video Producer. Simplify the following technical concept (Zero-Trust Network Security) into a conversational explainer script for non-technical executives.

### Script Requirements:
1. **The Problem**: Open by agitating the pain point: Corporate data breaches because employees use the same password everywhere.
2. **The Analogy**: Break down the complexity using a simple, real-world analogy related to a high-security hotel keycard system.
3. **The Pitch**: Transition into introducing our solution: SentinelGate.
4. **The Tone**: Ensure the language is friendly and reassuring. Strip out technical jargon like 'IPsec', 'Subnets', and 'Hash algorithms'.
5. **Call to Action**: End with a direct call to action to sign up for a free security audit.

### Output Constraints:
Write paragraphs measuring 2-3 sentences max. Aim for an exact reading time of 60 seconds (approx 140 words). Format entirely for a voiceover talent to read seamlessly.
```
