# Template: Engaging Email Newsletter Headers

Use this template to generate wide-format, high-visibility header images for newsletters. It is designed to create a "Visual Hook" that sets the mood for the rest of the email, utilizing negative space strategically to allow you to overlay your logo or title without creating a cluttered or unreadable design.

## 📋 Prompt Template

```text
Act as a Digital Graphic Designer and E-mail Marketing Expert. Generate a wide-aspect ratio (3:1 or 4:1) header image prompt for a newsletter about [NEWSLETTER_THEME].

### Design Context:
- **Newsletter Goal**: [e.g., Weekly tech roundup, Lifestyle inspiration, eCommerce flash sale].
- **Brand Vibe**: [e.g., Clean & Corporate, Vibrant & Fun, Moody & Artistic].

### Visual Requirements:
1. **Aspect Ratio & Framing**: Specify a [WIDE_PANORAMIC] view. Design the focal point to be on the [LEFT_OR_RIGHT] side.
2. **The "Text Safe" Area**: Describe a [MINIMALIST_NEGATIVE_SPACE] section (e.g., "A solid block of muted color on the left," "A blurred landscape with no busy details").
3. **The Subject Matter**: Represent the [NEWSLETTER_THEME] using [ABSTRACT_OR_LITERAL_OBJECTS] (e.g., "A neat stack of futuristic tech gadgets," "A peaceful morning coffee table with a newspaper").
4. **Color Palette**: Specify 2-3 [BRAND_COLORS] that should dominate the scene.
5. **Lighting & Texture**: Define the [ATMOSPHERE] (e.g., "Bright morning sunlight with sharp shadows," "Soft diffused studio lighting," "A flat, vector-style illustrative look").

### Output Constraints:
- Use a Clean, Functional, and Modern tone.
- Avoid busy patterns that make text unreadable.
- Provide the final prompt in a copy-pasteable code block.
```

## 🧩 Variables to Fill Out

- `[NEWSLETTER_THEME]` - The subject (e.g., AI and Robotics, Healthy Cooking, Minimalist Living).
- `[LEFT_OR_RIGHT]` - Where the main object should be, allowing the opposite side for text.
- `[MINIMALIST_NEGATIVE_SPACE]` - e.g., "Large area of white space," "Deep blue gradient."
- `[ABSTRACT_OR_LITERAL]` - Symbols vs real items.
- `[BRAND_COLORS]` - The colors that match your brand's existing design.

## 💡 Pro-Tips

- **The "Rule of Negative Space":** In a header, the image shouldn't compete with the title. Tell the AI to "Desaturate or blur the area where the title 'The [INSERT_NAME] Weekly' will be placed."
- **Consistency is Key:** Use the same [STYLE_SPECIFIER] (e.g., "3D Claymation," "Bento box style," "Macro photography") for every edition to build a recognizable visual brand.
- **Mobile First:** Email headers are small on phones. Ensure the [FOCAL_POINT] is bold and clear, even when scaled down to 300px wide.
