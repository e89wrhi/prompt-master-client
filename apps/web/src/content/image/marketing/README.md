# 📈 Marketing Image Prompts

Templates designed to generate strictly commercial, high-conversion visual assets for ads and landing pages.

## Available Categories

- [**Ad Campaign Visuals**](./ad_campaign_visuals/index.md): High-contrast, scroll-stopping Facebook/Instagram ad images.
- [**Email Newsletter Headers**](./email_newsletter_headers/index.md): Wide, negative-space heavy banners for text overlays.
- [**Landing Page Hero Images**](./landing_page_hero_images/index.md): Modern, flat-vector, or 3D illustration assets for SaaS websites.
- [**YouTube Thumbnails**](./youtube_thumbnails/index.md): Hyper-exaggerated expressions and bright neon background parameters.
