# Style: Vibrant Anime & Manga

![Anime](/images/styles/anime.png)

This style ranges from the cinematic, hyper-detailed backgrounds of Studio Ghibli to the sleek, modern line-work of high-octane action anime. It emphasizes flat cell-shading, vibrant color palettes, dramatic speed lines, and expressive character designs.

## 📋 Prompt Template

```text
Act as an Anime Concept Artist and Lead Animator. Generate a prompt for a [SCENE/CHARACTER] in the [ANIME_STUDIO/STYLE] style.

### Artistic Direction:
- **Style Archetype**: [e.g., Studio Ghibli, Makoto Shinkai, 90s Cyberpunk, Modern Shonen].
- **Line Work**: [e.g., Sharp and clean, soft and hand-drawn, thick black outlines].
- **Shading**: [e.g., Flat cell-shading, soft gradients, dramatic cross-hatching].

### Visual Requirements:
1. **The Subject**: Describe [SUBJECT] with focus on anime-specific traits (e.g., "Large expressive eyes," "Dynamic wind-blown hair," "Intricate tech-suit").
2. **The Environment**: Define the background (e.g., "Lush, hand-painted watercolor clouds," "A neon-soaked cyberpunk street").
3. **Action & Energy**: Use [SPEED_LINES] or [MAGIC_PARTICLES] to create movement.

### Output Constraints:
- Vibrant and High-Energy tone.
- Conclude with "High-definition anime, cell-shaded, masterwork composition."
```

## 🧩 Variables to Fill Out

- `[ANIME_STUDIO]` - e.g., Kyoto Animation, Studio Wit, MAPPA.
- `[SUBJECT]` - Character or Landscape.
- `[SHADING]` - The key to the anime look.

## 💡 Pro-Tips

- **Makoto Shinkai Lighting:** If you want that "glowing" look, ask for "Cinematic lighting with lens flare and detailed reflection in the eyes."
- **Ghibli Texture:** For a cozy feel, ask for "Hand-painted gouache backgrounds with soft, nostalgic lighting."
- **Focus on the Eyes:** In anime, eyes are the window to the soul. Ask for "Highly detailed irises with a starry reflection."
