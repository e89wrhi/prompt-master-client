# Style: Modern Isometric 3D

![Modern Isometric](file:///d:/dev/_projects/prompt-library/assets/images/modern_isometric.png)

This style utilizes a fixed 45-degree angle (No perspective distortion) to create "Self-Contained" worlds, layouts, or objects. It emphasizes clean geometric shapes, soft ambient shadows (Ambient Occlusion), pastel color palettes, and a "Bento Box" organization.

## 📋 Prompt Template

```text
Act as a Visual Storyteller and Isometric 3D Illustrator. Generate a prompt for an isometric [SCENE/OBJECT] on a [FLOATING_BASE].

### Illustrative Specs:
- **Geometry**: [e.g., Low-poly, clean vector 3D, soft clay].
- **Angle**: Perfect 45-degree isometric view (no perspective).
- **Palette**: [e.g., Soft Pastel, Vibrant Synthwave, Monochrome White].

### Visual Requirements:
1. **The Scene**: Describe the [SUBJECT] contained on a square or circular base (e.g., "A miniature smart city," "A developer's desk," "An island in the sky").
2. **Details**: Specify the [MICRO-ELEMENTS] (e.g., "Tiny glowing dots," "Small green bushes," "Micro-tech components").
3. **Lighting**: Define the [SHADOWS] (e.g., "Soft global illumination," "Ambient occlusion in the creases").

### Output Constraints:
- Satisfying, Clean, and Precise tone.
- Conclude with "Cinema 4D style, isometric 3D, clean edges, matte finish."
```

## 🧩 Variables to Fill Out

- `[SCENE/OBJECT]` - The miniature world.
- `[FLOATING_BASE]` - Square, Circle, or Diamond.
- `[MICRO-ELEMENTS]` - The tiny things that make it feel alive.

## 💡 Pro-Tips

- **Ambient Occlusion is Crucial:** For the "premium 3D" feel, always ask for [AMBIENT_OCCLUSION]. It adds the soft shadows in corners that give 3D depth.
- **Bento Box Layout:** Use "contained within sections" to organize multiple unrelated items into a cohesive isometric grid.
- **Texture Contrast:** Use "Matte plastic objects on a smooth glass base" for a modern tech aesthetic.
