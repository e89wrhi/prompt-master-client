# Examples: Modern Isometric 3D

## 1. Smart-Tech Bedroom

```text
Act as an Isometric Illustrator. Generate a prompt for a "Futuristic Bedroom."

### Illustrative Specs:
- **Geometry**: Clean vector 3D models.
- **Angle**: Perfect 45-degree isometric on a floating white square base.
- **Palette**: Soft Mint Green, White, and Oak.

### Visual Requirements:
- A bed with a glowing smart-lamp, a small robot vacuum on the floor, and a holographic window view.
- Soft global illumination and ambient occlusion.

### Output Constraints:
- Cinema 4D style, isometric 3D, matte finish.
```

## 2. Cloud Server Infrastructure

```text
Act as a Visual Storyteller. Generate an isometric view of a server room.

### Illustrative Specs:
- **Palette**: Neon Blue and Charcoal.

### Visual Requirements:
- Floating dark-grey platform.
- Rows of server racks with tiny glowing blue dots representing data flow.

### Output Constraints:
- Clean edges, precise tone.
```
