# Examples: Modern Minimalist Design

## 1. Single Office Chair

```text
Act as a Minimalist Designer. Generate a prompt for a single piece of furniture.

### Design Principles:
- **Palette**: Off-white and light Oak wood.
- **Negative Space**: 80% empty white studio space.
- **Lighting**: Soft diffused natural light from the side.

### Visual Requirements:
- A single Eames-style chair in the dead center of the frame.
- Smooth matte shell and fine-grain wood legs.

### Output Constraints:
- Minimalist design, clean lines, 8k.
```

## 2. Abstract Geometry

```text
Act as a Graphic Designer. Generate a prompt for an abstract minimalist poster.

### Design Principles:
- **Palette**: Charcoal Black and a single Gold dot.
- **Negative Space**: Large black void.

### Visual Requirements:
- A single perfectly circular gold leaf dot in the bottom-right.
- Subtle paper texture background.

### Output Constraints:
- High negative space, sophisticated tone.
```
