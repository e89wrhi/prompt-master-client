# Style: Modern Minimalist Design

![Minimalist](/images/styles/minimalist.png)

This style is defined by the philosophy of "Less is More." It emphasizes clean lines, vast negative space, a limited color palette (Monochrome/Neutrals), soft lighting, and a single, strong focal point to create a sense of calm and sophistication.

## 📋 Prompt Template

```text
Act as a Minimalist Graphic Designer and UI/UX Specialist. Generate a prompt for a [SUBJECT/SCENE].

### Design Principles:
- **Palette**: [e.g., Off-white and Beige, Monochrome Black, Soft Muted Greys].
- **Negative Space**: [e.g., 70% empty space, large white void, single center object].
- **Lighting**: [e.g., Soft diffused natural light, single sharp spotlight, shadowless global illumination].

### Visual Requirements:
1. **The Focal Point**: Describe [SUBJECT] with simple, essential geometric shapes.
2. **Composition**: Specify the [POSITION] (e.g., "Dead center," "Rule of thirds bottom-right").
3. **Texture**: Define the [SURFACE] (e.g., "Smooth matte," "Coarse linen," "Fine-grain paper").

### Output Constraints:
- Calm, Clean, and Sophisticated tone.
- Conclude with "Minimalist design, high negative space, clean lines, 8k."
```

## 🧩 Variables to Fill Out

- `[SUBJECT]` - The single item of focus.
- `[NEGATIVE_SPACE]` - How "Empty" should it be?
- `[PALETTE]` - Usually 1-2 colors.

## 💡 Pro-Tips

- **Avoid the Clutter:** If the prompt starts getting too long, you're moving away from minimalism. Delete adjectives that don't contribute to the [SILHOUETTE].
- **Soft Shadows Matter:** To prevent a minimalist shot from looking "flat," ask for "Soft, long ambient shadows" to add 3D depth.
- **Micro-Texture:** A solid white background can be boring. Ask for "Subtle paper texture" or "Soft limestone grain" to add physical realism.
