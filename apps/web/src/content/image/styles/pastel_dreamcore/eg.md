# Examples: Ethereal Pastel Dreamcore

## 1. The Endless Indoor Pool

```text
Act as a Dreamcore Architect. Generate a prompt for a "Liminal Pool" scene.

### Dreamscape Profile:
- **Palette**: Soft Cyan and Peach.
- **Atmosphere**: Thick hazy mist over the water.
- **Sub-Genre**: Liminal space (Empty indoor pool).

### Visual Requirements:
- A single white tiled pool that stretches infinitely into a pink and teal fog.
- Soft-focus dreaminess.

### Output Constraints:
- Pastel dreamcore aesthetic, nostalgic surrealism.
```

## 2. Floating Payphone in the Clouds

Act as a Surrealist Video Artist. Generate a prompt for a nostalgic object.

### Dreamscape Profile:

- **Concept**: A single retro payphone floating in a sea of puffy purple clouds.

### Output Constraints:
