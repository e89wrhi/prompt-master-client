# Examples: Professional & Stylized Avatars

## 1. Professional Tech Lead (Photorealistic)

```text
Act as a Personal Brand Consultant. Generate a prompt for a photorealistic profile picture for a Senior Software Engineer.

### Brand Persona:
- **Vibe**: Expert Tech Lead, Trustworthy and Modern.
- **Industry**: Software Engineering.

### Visual Requirements:
1. **Framing**: Tight shoulder-up portrait, straight eye-level gaze.
2. **Background**: A high-contrast deep charcoal grey with a subtle blue rim-light.
3. **Lighting**: Soft wrap-around studio lighting with distinct catch-lights in the eyes.
4. **Features**: Black turtleneck, thin-framed modern glasses, well-groomed beard.
5. **Technical**: 85mm prime lens, F/1.8, razor-sharp focus on the eyes, blurred background.

### Output Constraints:
- Professional and Charismatic tone.
- Copy-pasteable code block.
```

## 2. Creative "Pixar-Style" Blogger (Stylized)

```text
Act as an Avatar Designer. Generate a stylized 3D avatar for a wellness blogger.

### Brand Persona:
- **Vibe**: Friendly, Vibrant, and Approachable.

### Visual Requirements:
- **Style**: 3D Pixar-style animation render.
- **Background**: Solid, bright safety-orange (High visibility).
- **Features**: A character with a warm smile, wearing a green hoodie, with soft curly hair.
- **Lighting**: Bright, shadowless butterfly lighting.

### Output Constraints:
- High-Visibility Boldness.
```
