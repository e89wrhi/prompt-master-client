# Examples: High-Visibility Twitch Emotes

## 1. "Cyber-Cat" Emote Set (Chibi Style)

```text
Act as an Emote Artist. Generate a set of 3 "Cyber-Cat" emotes.

### Emote Context:
- **Character**: A small robotic cat with glowing blue visor eyes.
- **Style**: Kawaii Chibi.

### Visual Requirements:
1. **Emotions**: 
   - HYPE: Cat with glowing red visor and lightning bolts in background.
   - SAD: One drooping circuit ear and a "404" error on the visor.
   - LOVE: Visor eyes turned into pink hearts.
2. **Readability**: Thick black outlines, limited 3-color palette (Neon Blue, Black, White).
3. **Focus**: Head takes up 80% of the canvas; large expressive eyes.
4. **Technical**: Clean cell-shading, pure white background.

### Output Constraints:
- High-Energy and Fun tone.
- Copy-pasteable code block.
```

## 2. "Founder GG" Emote (90s Cartoon)

```text
Act as a Streamer Brand Designer. Generate a "GG" emote for a tech founder.

### Emote Context:
- **Character**: A stylized cartoon version of a guy with glasses and a beard.
- **Style**: Bold 90s Cartoon.

### Visual Requirements:
- **Expression**: "GG" - The character is wearing a giant golden crown and giving two thumbs up.
- **Features**: Large "GG" golden letters floating above the head.
- **Readability**: High contrast between the yellow crown and black hair.

### Output Constraints:
- Icon-Focused tone.
```
