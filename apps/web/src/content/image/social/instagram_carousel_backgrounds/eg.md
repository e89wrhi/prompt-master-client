# Examples: High-Engagement Instagram Backgrounds

## 1. "5 Tips for Financial Freedom" (Professional & Academic)

```text
Act as a Visual Brand Designer. Generate a set of 5 backgrounds for a financial education carousel.

### Design Context:
- **Tone**: Academic, Trustworthy, and Clean.
- **Texture**: Heavy matte paper with subtle "noise" grain.

### Visual Requirements:
1. **Flow**: Each slide features a thin, professional gold line that moves higher up the page as the user swipes.
2. **Zones**: The bottom 30% of each slide is a solid Dark Navy block for "Fact" callouts; the rest is white paper.
3. **Palette**: Dark Navy, Gold, and Bone White.
4. **Technical**: Minimalist vector style, sharp 2D edges, flat aesthetic.

### Output Constraints:
- Cohesive and Modern tone.
- Copy-pasteable code block.
```

## 2. "Modern Creative Workflow" (Vibrant & High-Energy)

```text
Act as a Social Media Strategist. Generate backgrounds for a "Creativity Boost" carousel.

### Design Context:
- **Tone**: High-Energy and Vibrant.
- **Texture**: Sleek glassmorphism / 3D gradients.

### Visual Requirements:
- **Flow**: Floating 3D "Glass" bubbles that seem to move further into the background as the slides progress.
- **Zones**: Centered text area with a blurred "frosted glass" rectangle for readability.
- **Palette**: Electric Purple, Hot Pink, and Cyan.

### Output Constraints:
- Continuous Journey feel.
```
