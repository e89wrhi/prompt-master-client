# 📱 Social Media Image Prompts

Specialized prompts for platforms that require specific aesthetic trends, dimensions, and engagement factors.

## Available Categories

- [**Pinterest Infographic Assets**](./pinterest_infographic_assets/index.md): Tall, highly visual, aesthetic listicle graphics.
- [**Profile Picture Avatars**](./profile_picture_avatars/index.md): Anime style, corporate headshots, or stylized vector portraits.
- [**Twitch Emotes**](./twitch_emotes/index.md): Simplistic, bold-lined reactions suitable for heavy downscaling.
