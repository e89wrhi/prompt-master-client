# 🎨 Artistic Image Prompts

Templates focused purely on aesthetic, conceptual, and illustrative generations using tools like Midjourney, DALL-E, and Stable Diffusion.

## Available Categories

- [**Abstract Concepts**](./abstract_concepts/index.md): Prompts for generating moody, non-literal, and surreal imagery.
- [**Cyberpunk Aesthetics**](./cyberpunk_aesthetics/index.md): Neon lighting, rain-slicked streets, and dystopian tech styles.
- [**Fantasy Landscapes**](./fantasy_landscapes/index.md): Epic mountains, glowing forests, and castles with specific camera ratios.
- [**Oil Painting Styles**](./oil_painting_styles/index.md): Emulating Renaissance, Impressionism, or specific classic artists.
