# Examples: Strategic Dashboard Wireframes

## 1. Fintech Performance Dashboard (High Density)

```text
Act as a Senior UX Designer. Generate a wireframe for a high-density Fintech stock-trading dashboard.

### UX Objective:
- **Core Goal**: Real-time monitoring of multiple stock tickers and portfolio value.
- **Density**: High-density/info-heavy.

### Layout Requirements:
1. **Navigation**: Collapsible left sidebar for "Portfolio," "Watchlist," and "Settings."
2. **Main Grid**: A 4-column Bento-box grid.
3. **Data Blocks**:
   - Large candlestick chart taking up 50% of the screen.
   - Small list of "Recent Alerts" on the far right.
   - 4 horizontal KPI cards at the top (Total Balance, Daily Gain, etc.).
4. **Style**: Low-fidelity grayscale; using varying shades of grey to denote priority.

### Output Constraints:
- Logical and Structural tone.
- Copy-pasteable code block.
```

## 2. Smart Home Control Hub (Clean & Spacious)

```text
Act as a Product Manager. Generate a wireframe for a tablet-based Smart Home controller.

### UX Objective:
- **Core Goal**: Quick access to "Lights," "Temperature," and "Security."
- **Density**: Clean and spacious.

### Layout Requirements:
- **Navigation**: Minimal top bar with "Rooms" dropdown.
- **Main Grid**: Large 3x3 grid of icons representing different devices.
- **Controls**: Large slider for brightness on the left; one large "Master Off" button at the bottom.

### Output Constraints:
- Precise and Minimalist tone.
```
