# Examples: Professional App Icons

## 1. Productivity App: "FocusHour" (Glassmorphism)

```text
Act as a Senior Icon Designer. Generate an app icon for a productivity timer called "FocusHour."

### Icon Strategy:
- **Function**: Deep focus work timer.
- **Style**: Apple-style Glassmorphism.

### Visual Requirements:
1. **Metaphor**: A stylized, minimalist hourglass.
2. **Material**: The top half of the hourglass is frosted glass; the bottom is filled with glowing golden "sand" particles.
3. **Background**: A rounded square with a deep midnight-blue to charcoal gradient.
4. **Lighting**: Soft inner glow from the sand; subtle rim-lighting on the glass edges.
5. **Technical**: 8k, sharp edges, clean silhouette.

### Output Constraints:
- Iconic and Modern tone.
- Copy-pasteable code block.
```

## 2. Crypto Vault: "EtherSafe" (Modern Skuomorphism)

```text
Act as a Brand Architect. Generate an icon for a secure crypto wallet.

### Icon Strategy:
- **Function**: Secure digital vault.
- **Style**: High-tech Modern Skuomorphism.

### Visual Requirements:
- **Metaphor**: A heavy, hexagonal mechanical lock.
- **Material**: Brushed titanium and dark carbon-fiber.
- **Background**: Solid vibrant electric-blue.
- **Lighting**: Hard side-lighting that highlights the metallic texture.

### Output Constraints:
- High-Resolution and Bold tone.
```
