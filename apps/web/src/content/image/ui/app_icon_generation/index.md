# Template: Professional App Icon Generation

Use this template to generate a high-impact, recognizable icon for mobile or desktop applications. It is designed to navigate the "One Metaphor" rule—distilling your app's core function into a single, bold visual symbol utilizing [VIBRANT_GRADIENTS], [TACTILE_SHADOWS], and [ICONOGRAPHIC_SIMPLICITY].

## 📋 Prompt Template

```text
Act as a Senior Icon Designer and Brand Architect. Generate a high-fidelity app icon prompt for [APP_NAME].

### Icon Strategy:
- **Core Function**: [e.g., A task manager, a fitness tracker, a secure vault].
- **Style Archetype**: [e.g., Apple-style Glassmorphism, Modern Skuomorphism, Flat Neon, Google Material].

### Visual Requirements:
1. **The Core Metaphor**: Define the [SINGLE_OBJECT] that represents the app (e.g., "A stylized golden hourglass," "A glowing blue shield," "A minimalist geometric flame").
2. **Surface & Material**: Specify the [PHYSICAL_FEEL] (e.g., "Frosted glass with inner luminosity," "Brushed titanium with tactile ridges," "High-gloss liquid plastic").
3. **Background & Contrast**: Describe the [CONTAINER_BACKGROUND] (e.g., "A deep navy rounded square," "A vibrant sunset gradient," "A clean white void with soft ambient shadows").
4. **Lighting & Shadow Play**: Define the [DIRECTIONAL_LIGHTING] (e.g., "Soft top-down light creating a subtle drop shadow," "Inner glow emanating from the core," "Rim-lighting that defines the icon's depth").
5. **Technical Fidelity**: Conclude with fidelity cues (e.g., "Sharp vector edges, 8k resolution, Unreal Engine 5 aesthetic, clean silhouette, recognizable at small scale").

### Output Constraints:
- Use an Iconic, Modern, and High-Resolution tone.
- Avoid text within the icon; focus entirely on the [SYMBOL].
- Provide the final prompt in a copy-pasteable code block.
```

## 🧩 Variables to Fill Out

- `[APP_NAME]` - The name of the project.
- `[SINGLE_OBJECT]` - The #1 metaphor for your app.
- `[STYLE_ARCHETYPE]` - The visual design language.
- `[PHYSICAL_FEEL]` - The texture that gives it depth.
- `[SYMBOL]` - The refined final shape.

## 💡 Pro-Tips

- **The "One Metaphor" Rule:** An icon shouldn't tell a story; it should be a [NOUN]. If your app is for "Cooking with Friends," don't show people cooking. Show a [WHISK] or a [CHEF_HAT].
- **Gradients add "Newness":** To make an icon look modern, use [MULTI-STOP_GRADIENTS] instead of solid colors. Ask the AI to "Infuse a 3-step gradient (e.g., Teal to Royal Blue to Deep Purple) into the core symbol."
- **Focus on the Silhouette:** A good icon is recognizable by its shape alone. Ask the AI to "Prioritize a bold, high-contrast silhouette that is distinguishable at 64x64px."
