# Template: High-Fidelity Web Design Mockups

Use this template to generate modern, trending website layouts for portfolios, client pitches, or development inspiration. It is designed to navigate specific [UI_TRENDS]—such as Bento grids, Glassmorphism, or Dark Mode—utilizing [TYPOGRAPHICAL_HIERARCHY] and [CLEAN_INTERACTION_MODELS] to create professional-grade mockups.

## 📋 Prompt Template

```text
Act as a Senior Web Designer and UI Strategist. Generate a high-fidelity landing page mockup for [SITE_NAME], a [INDUSTRY] company.

### Design Strategy:
- **Core Aesthetic**: [e.g., Ultra-minimalist Swiss, Bold 90s Brutalist, Sleek Silicon Valley Dark-Mode, Pastel Playful].
- **UI Trend Integration**: [e.g., Bento box layout, large-radius borders, frosted glass elements, thin-line borders].

### Visual Requirements:
1. **Layout Architecture**: Describe the [SECTIONAL_FLOW] (e.g., "A sprawling hero section followed by a 3-column feature grid," "A vertical split-screen with an image on the left and text on the right").
2. **Typography Hierarchy**: Specify the [FONT_STYLING] (e.g., "Large, bold serif headers with clean monospace sub-text," "Modern geometric sans-serif throughout with high letter-spacing").
3. **Imagery & Staging**: Define the [VISUALS] used (e.g., "Abstract 3D glass shapes," "High-end editorial lifestyle photography," "Clean flat-design illustrations").
4. **Color & Contrast**: Specify a [SPECIFIC_PALETTE] and [LUMINOSITY_MODE] (e.g., "Deep charcoal backgrounds with electric-purple accents," "Bright airy whites with soft-grey shadows").
5. **Interactive Cues**: Describe the [BUTTONS_AND_FORMS] (e.g., "Glowing CTA buttons with soft drop shadows," "Clean minimalist input fields with thin borders").

### Output Constraints:
- Use a Modern, Professional, and Aesthetic tone.
- Ensure the Mockup accounts for [DESKTOP_OR_MOBILE] view.
- Provide the final prompt in a copy-pasteable code block.
```

## 🧩 Variables to Fill Out

- `[SITE_NAME]` - The brand.
- `[INDUSTRY]` - e.g., FinTech, Sustainable Fashion, AI Startup, Personal Portfolio.
- `[SECTIONAL_FLOW]` - The "Scroll" journey.
- `[DESKTOP / MOBILE]` - The responsive target.
- `[SPECIFIC_PALETTE]` - The color identity.

## 💡 Pro-Tips

- **Bento Boxes solve Busy-ness:** If the website has a lot of small features to show, tell the AI to "Design the features section using a [BENTO_BOX_LAYOUT] with variable-sized cards."
- **The "Border" Trend:** For a very modern look, ask for [THIN_LINE_BORDERS] (1px) in a slightly lighter shade than the background. It adds a technical, organized feel to the UI.
- **Micro-Copy Matters:** Even in a mockup, the text placeholders matter. Ask the AI to "Include clear H1 and H2 headers that represent the [SITE_NAME] value proposition."
