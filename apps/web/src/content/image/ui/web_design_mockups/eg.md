# Examples: High-Fidelity Web Mockups

## 1. AI Startup: "NeuroFlow" (Silicon Valley Dark Mode)

```text
Act as a Senior Web Designer. Generate a landing page mockup for "NeuroFlow," an AI-native workspace.

### Design Strategy:
- **Aesthetic**: Sleek Silicon Valley Dark-Mode.
- **Trend**: Bento box feature grid and large-radius borders.

### Visual Requirements:
1. **Layout**: A wide hero section with a 3D glowing "brain" icon, followed by a Bento-box grid of 6 feature cards.
2. **Typography**: Clean geometric sans-serif (Inter) with high letter-spacing.
3. **Imagery**: Abstract 3D glass shapes floating in the background.
4. **Palette**: Deep Charcoal (#1A1A1A), Electric Purple (#8A2BE2), and Cyber Lime accents.
5. **Interactive**: Glowing "Start Free Trial" button with a soft purple shadow.

### Output Constraints:
- Modern and Professional tone.
- Desktop view mockup.
```

## 2. Sustainable Fashion: "EarthWear" (Minimalist Swiss)

```text
Act as a UI Strategist. Generate a homepage for an eco-friendly clothing brand.

### Design Strategy:
- **Aesthetic**: Ultra-minimalist Swiss.
- **Trend**: High-end editorial photography and huge white space.

### Visual Requirements:
- **Layout**: Full-screen hero image of a person in a linen shirt in a field, with the navigation bar in a thin-line border at the top.
- **Typography**: Large bold serif "EARTHWEAR" centered; small clean sans-serif sub-text.
- **Palette**: Off-white, Sage Green, and Slate Grey.

### Output Constraints:
- High-fidelity aesthetic.
```
