# 🖥️ UI & Interface Image Prompts

Templates leveraging image generators to brainstorm user experiences, layouts, and interface elements.

## Available Categories

- [**App Icon Generation**](./app_icon_generation/index.md): Skeuomorphic, neumorphic, and flat-design mobile app icons.
- [**Dashboard Wireframes**](./dashboard_wireframes/index.md): SaaS analytics layouts, charts, and dark-mode data visualizations.
- [**Game UI Assets**](./game_ui_assets/index.md): Fantasy inventory slots, sci-fi HUDs, and stylized progress bars.
- [**Web Design Mockups**](./web_design_mockups/index.md): Generating overarching dribbble-style inspiration layouts.
