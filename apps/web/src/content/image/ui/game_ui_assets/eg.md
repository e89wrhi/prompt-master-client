# Examples: High-Detail Game UI Assets

## 1. Dark Fantasy RPG Menu (Gothic Style)

```text
Act as a Senior Game UI Artist. Generate a prompt for a character inventory panel.

### Game World:
- **World Vibe**: Gritty dark fantasy / Gothic.
- **Core Materials**: Weathered obsidian stone and aged parchment.

### Asset Requirements:
1. **Asset**: A 5x4 inventory grid encased in a stone frame.
2. **Details**: Iron-wrought trim with glowing red runes at each corner. The parchment background has subtle blood-splatter textures.
3. **States**: Highlighting an item slot creates a soft white "ghostly" glow behind the icon.
4. **Color Schema**: Deep Crimson, Charcoal, and Bone White.
5. **Technical**: Unreal Engine 5 aesthetic, sharp stone textures, transparent-ready background.

### Output Constraints:
- Imaginative and System-Focused tone.
- Copy-pasteable code block.
```

## 2. Sci-Fi FPS HUD (Futuristic Light)

```text
Act as a Visual Designer. Generate a HUD for a futuristic shooter.

### Game World:
- **World Vibe**: Sleek futuristic sci-fi / Cyberpunk.
- **Core Materials**: Translucent holographic light and brushed chrome.

### Asset Requirements:
- **Asset**: A circular health and ammo counter that surrounds the player's crosshair.
- **Details**: Floating digital noise and "glitch" artifacts along the edges of the HUD.
- **States**: Health below 20% turns the entire HUD into a flickering hazard-red.

### Output Constraints:
- Cohesive UI Kit feel.
```
