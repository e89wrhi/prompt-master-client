# Template: Clean Isometric & "Bento Box" Illustrations

Use this template to generate precise, clean, and visually satisfying isometric illustrations. It is designed to visualize complex systems—like cities, code infrastructures, or office layouts—using a perfect 45-degree angle, [CLEAN_GEOMETRIC_EDGES], and a [MINIMALIST_PALETTE].

## 📋 Prompt Template

```text
Act as a Visual Storyteller and Isometric Illustrator. Generate a high-detail isometric 3D illustration for [SUBJECT_SYSTEM].

### Illustrative Context:
- **Concept**: [e.g., A smart-tech apartment, a blockchain transaction flow, a peaceful library].
- **Style Variation**: [e.g., Low-poly game aesthetic, Clean vector 3D, Soft "Claymation" style].

### Visual Requirements:
1. **Geometry & Angle**: Maintain a [PERFECT_45_DEGREE_ISOMETRIC] view. Ensure [NO_PERSPECTIVE_DISTORTION] (parallel lines).
2. **The "Bento Box" Layout**: Describe the [SEGREGATED_COMPARTMENTS] (e.g., "The scene is contained within a clean floating square platform," "Each element has its own distinct island").
3. **Color & Lighting**: Specify a [SOFT_PASTEL_OR_VIBRANT] palette. Use [GLOBAL_ILLUMINATION] for soft, shadowless lighting.
4. **Details & Micro-Elements**: Describe the [TINY_DETAILS] that make it feel alive (e.g., "Tiny glowing dots moving along wires," "Small green plants in windows," "Microscopic 3D icons").
5. **Technical Specs**: Conclude with fidelity cues (e.g., "Cinema 4D style, clean edges, matte finish, soft ambient occlusion, high resolution, solid pastel background").

### Output Constraints:
- Use a Precise, Clean, and Satisfying tone.
- Ensure the scene feels "Self-Contained" on its own floating base.
- Provide the final prompt in a copy-pasteable code block.
```

## 🧩 Variables to Fill Out

- `[SUBJECT_SYSTEM]` - e.g., A developer's workspace, A sustainable farm, A cloud server rack.
- `[LOW_POLY / CLAYMATION]` - The specific visual "vibe."
- `[TINY_DETAILS]` - The "Easter eggs" that add value.
- `[PASTEL / VIBRANT]` - The color personality.

## 💡 Pro-Tips

- **Ambient Occlusion is the Secret:** For that "premium 3D" look, include [AMBIENT_OCCLUSION] in the prompt. It adds those soft, realistic shadows in the corners and creases of the 3D shapes.
- **The "Floating Island" Hook:** Always specify that the illustration is on a [FLOATING_BASE]. This prevents it from feeling like a standard wide shot and makes it feel like an architectural model.
- **Color Coding:** Use colors to represent data or flow (e.g., "Blue pipes for data inflow, Green pipes for outflow").
