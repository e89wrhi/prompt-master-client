# Examples: Clean Isometric Illustrations

## 1. Smart-Home Living Room

```text
Act as an Isometric Illustrator. Generate a 3D illustration of a "Smart Minimalist Living Room."

### Illustrative Context:
- **Concept**: A tech-integrated cozy apartment.
- **Style**: Soft "Claymation" / Matte 3D.

### Visual Requirements:
1. **Geometry**: Perfect 45-degree isometric view on a floating white square base.
2. **Layout**: One open-plan room with a sofa, a bookshelf, and a large window.
3. **Color**: Pastel Mint Green, Soft Grey, and Light Oak wood.
4. **Details**: A tiny robot vacuum on the floor, a single glowing smart-lamp, and many small potted succulents.
5. **Technical**: Cinema 4D style, soft ambient occlusion, matte finish.

### Output Constraints:
- Precise and Satisfying tone.
- Copy-pasteable code block.
```

## 2. Cloud Data Infrastructure "Bento Box"

```text
Act as a Visual Storyteller. Generate an illustration for "The Data Flow."

### Illustrative Context:
- **Concept**: A server room where data is being processed into gold cubes.

### Visual Requirements:
- **Style**: High-contrast, vibrant 3D vector.
- **Layout**: Floating dark-grey platform.
- **Details**: Rows of glowing server racks on one side, a conveyor belt in the middle moving blue data spheres, and a "processing" machine turning them into gold cubes on the other side.
- **Palette**: Neon Blue, Gold, and Charcoal.

### Output Constraints:
- Clean and Strategic tone.
```
