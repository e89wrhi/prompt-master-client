# Template: Professional Packaging & Label Design

Use this template to generate high-fidelity concepts for product packaging. It is designed to specify the [STRUCTURE] of the package, the [TYPOGRAPHICAL_LAYOUT], and the [TACTILE_FINISHES] (e.g., gold foil, spot UV, matte cardboard) required to create a "luxury unboxing" experience or a standout retail shelf presence.

## 📋 Prompt Template

```text
Act as a Senior Brand Designer and Packaging Specialist. Generate a high-fidelity image prompt for the [PRODUCT_TYPE] packaging of [BRAND_NAME].

### Packaging Specs:
- **Format**: [e.g., 3D folding box, minimalist tube, artisanal glass bottle with label, eco-friendly paper wrap].
- **Core Aesthetic**: [e.g., Bold maximalist, Swiss modernist, vintage apothecary, playful organic].

### Visual Requirements:
1. **The Structural View**: Specify the [VIEW_ANGLE] (e.g., "Perspective shot of the closed 3D box," "Flat design layout of the un-folded label," "Stacked arrangement of 3 different box sizes").
2. **Typography & Layout**: Describe the [FONT_PERSONALITY] and [PLACEMENT] (e.g., "Large serif font centered on the front," "Tiny technical sans-serif text on the sides," "Bold handwritten script wrapping around the edges").
3. **Tactile Finishes & Materials**: Specify the [SURFACE_DETAILS] (e.g., "Matte black cardboard with gold foil embossed logo," "Recycled brown craft paper with a rough texture," "High-gloss spot UV coating on specific patterns").
4. **Color Story**: Specify 3 [STRATEGIC_COLORS] (e.g., Midnight Blue, Copper, and Cream).
5. **Background Atmosphere**: Describe the [STAGING_ENV] (e.g., "Resting on a clean white studio table," "Surrounded by the product's raw ingredients (e.g., cocoa beans)," "Suspended in a minimalist soft-grey space").

### Output Constraints:
- Use a Sharp, Detail-Oriented, and Brand-Focused tone.
- Conclude with technical photography cues (e.g., "Shot on 50mm macro, F/2.8, commercial product staging, hyper-realistic, 8k").
- Provide the final prompt in a copy-pasteable code block.
```

## 🧩 Variables to Fill Out

- `[PRODUCT_TYPE]` - e.g., Craft Chocolate, Premium Whiskey, CBD Oil, High-Tech Headphones.
- `[BRAND_NAME]` - Your name.
- `[FONT_PERSONALITY]` - Serif (Classic), Sans-Serif (Modern), Script (Human).
- `[SURFACE_DETAILS]` - e.g., Foil, embossing, gloss, matte.
- `[STRATEGIC_COLORS]` - The colors that represent the brand's price point.

## 💡 Pro-Tips

- **The "Hero Ingredient" Staging:** To add context, show the [RAW_INGREDIENTS] around the box (e.g., If it's a lemon-scented soap, place a half-cut, hyper-realistic lemon next to the packaging).
- **Embossing sells Luxury:** To make it look "Expensive," always include [FOIL_STAMPING] or [EMBOSSING]. It adds a 3D textural quality that suggests high production value.
- **Flat vs. 3D:** If you need to see the "Layout," ask for a [FLAT_DIELINE]. If you need to see the "Result," ask for a [3D_PERSPECTIVE_MOCKUP].
