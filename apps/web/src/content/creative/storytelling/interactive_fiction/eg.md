# Examples: Branching Narrative & Interactive Fiction

## 1. Horror: "The Ward at Midnight"

```text
Act as a Narrative Game Designer. I want to play a horror interactive story set in an abandoned 1950s hospital.

### Game Mechanics:
- **Choices**: 3 options per turn.
- **Variables**: Sanity, Battery Life (Flashlight), Inventory.
- **Goal**: Find your missing brother and escape.

### Requirements:
- Use Second-Person (YOU) prose.
- Provide a "Thematic Ending" if Sanity reaches zero.

### Output Constraints:
- Immersive and Reactive tone.
- Start with the first scene.
```

## 2. Fantasy Quest: "The Dragon's Bargain"

```text
Act as an Interactive Fiction Writer. I am a low-level diplomat sent to talk to a dragon.

### Mechanics:
- Variables: Dragon's Patience, Gold, Reputation.
- Special: Use hidden "Dragon's Patience" variable.

### Output Constraints:
- Meaningful consequences for every dialogue choice.
```
