# 🎬 Scriptwriting Templates

Frameworks designed for screenplays, stage plays, and YouTube scripts.

## Available Categories

- [**Character Arcs**](./character_arcs/index.md): The Hero's Journey, negative arcs, and character flaw generation.
- [**Dialogue Improvements**](./dialogue_improvements/index.md): Taking "on-the-nose" dialogue and adding subtext.
- [**Scene Outlines**](./scene_outlines/index.md): Beat sheets, Save The Cat milestones, and narrative pacing.
- [**World Building Rules**](./world_building_rules/index.md): Hard vs. Soft magic systems, economic structures, and fantasy pantheons.
