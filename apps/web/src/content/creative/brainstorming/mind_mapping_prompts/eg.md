# Examples: Structured Mind Mapping

## 1. Fantasy Novel World-Building

```text
Act as an Information Architect. Build a Mind Map for my epic fantasy world: "The Shattered Isles."

### Mind Map Structure:
1. **Central Node**: The Shattered Isles (A world of floating islands).
2. **Primary Branches**: 
   - Magic System (The "Aether" flow).
   - Politics (The Council of Seven).
   - Geography (The Void below).
   - History (The Great Sundering).
3. **Deep Dive**: List 3 sub-nodes for each (e.g., Under Magic: Aether-ships, Burnout, The Source).
4. **Cross-Links**: How the Aether flow (Magic) affects Geography (The Void).

### Output Constraints:
- Indented Markdown Outline.
- "The Missing Branch" included at the end.
```
