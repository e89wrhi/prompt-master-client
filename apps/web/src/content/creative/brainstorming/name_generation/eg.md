# Examples: Strategic Name Generation

## 1. Naming a "Deep-Sea Mining Robot"

```text
Act as a Naming Specialist. Generate 10 names for a deep-sea robotic explorer.

### Context:
- **Audience**: Marine biologists and high-tech investors.
- **Values**: Depth, Durability, Discovery.
- **Roots**: Abyss, Hydro, Naut.

### Naming Frameworks:
1. **Portmanteau**: AbyssBot, HydroScan.
2. **Roots**: Abysso, Nautis.
3. **Evocative**: Trench, Angler, Kraken.

### Output Constraints:
- Rationale for each name.
- Creative and Linguistic tone.
```

## 2. Naming an "Ethical AI Startup"

```text
Act as a Brand Consultant. Generate names for an AI transparency auditor.

### Context:
- **Values**: Trust, Clarity, Ethics.
- **Vibe**: Post-startup, professional, established.

### Output Constraints:
- Rate each on the "Radio Spelling Test."
- Analytical tone.
```
