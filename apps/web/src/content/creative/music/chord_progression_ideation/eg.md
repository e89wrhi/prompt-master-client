# Examples: Emotional Chord Progressions

## 1. "Lo-fi Chill" (Melancholic Warmth)

```text
Act as a Music Theorist. Design a progression for a Lo-fi Hip Hop track.

### Composition Context:
- **Emotion**: Melancholic warmth.
- **Key**: C Major.
- **Vibe**: Rain outside a window.

### Harmonic Framework:
1. **Sequence**: IVmaj7 - iii7 - ii7 - Imaj7.
2. **Voicing**: Use 9th extensions for every chord.
3. **Surprise**: Borrow a iv chord (Fm7) from the parallel minor.
4. **Rhythm**: "Lazy" timing, slightly behind the beat.

### Output Constraints:
- Technical and Precise tone.
- Conclude with Theory Tip.
```

## 2. "Epic Cinematic" (Heroic Tension)

```text
Act as a Professional Composer. Design a progression for a Hans Zimmer style score.

### Framework:
- Key: D Minor.

### Output Constraints:

```
