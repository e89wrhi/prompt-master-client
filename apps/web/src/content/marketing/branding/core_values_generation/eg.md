# Examples: Core Values Generation

## 1. Remote Software Startup ("AsyncFirst")

```text
Act as a Culture Consultant. Generate 5 Core Values for "AsyncFirst," a software company building tools for remote managers.

### Context & Mission:
We believe that work shouldn't be defined by 9-5 hours or constant meetings. We serve teams who want to reclaim their focus.

### Value Requirements:
1. **Action-Oriented Titles**: (e.g., "Write first, Walk later," "Outcome over Hours").
2. **Impact**: How it affects the team (less burnout) and customers (high product quality).
3. **The Non-Negotiable**: We will not hire people who "Need a meeting to decide everything."

### Output Constraints:
- Bold and Principles-First tone.
- Cultural Litmus Test included.
```

## 2. Sustainable Clothing Brand ("EarthFab")

```text
Act as a Brand Strategist. Generate values for "EarthFab."

### Context & Mission:
We make hoodies that last 10 years, using 0% new plastic. We serve eco-conscious outdoor enthusiasts.

### Value Requirements:
- Titles: "Endurance over Fashion," "Radical Traceability."
- The non-negotiable: We will not use a supplier if they can't show their water-recycling logs.

### Output Constraints:
- Authentic and Actionable tone.
```
