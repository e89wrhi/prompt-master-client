# Template: Actionable Core Values Generation

Use this template to move beyond generic, "corporate" buzzwords (like Integrity or Excellence) and define a set of unique core values that actually drive behavior. It is designed to create values that serve as a decision-making filter for your team and a brand pillar for your customers.

## 📋 Prompt Template

```text
Act as a Senior Culture Consultant and Brand Strategist. Generate 4-6 Core Values for [ORGANIZATION/PROJECT].

### Context & Mission:
[INSERT_WHAT_YOU_DO_AND_WHO_YOU_SERVE]

### Value Requirements:
1. **The "Value in Action" Test**: Don't just list a word. For each value, provide:
   - **The Title**: A short, punchy phrase (e.g., "[VALUE_NAME]").
   - **The Definition**: A 1-sentence explanation of what it means to us.
   - **The Behavior**: A concrete example of this value in action (e.g., "This means we [DO_X] even when it's [COSTLY/DIFFICULT]").
2. **Internal vs. External Impact**: Explain how this value impacts [THE_TEAM] vs. how it is perceived by [THE_CUSTOMER].
3. **The "Non-Negotiable" Rule**: Describe a situation where we would *refuse* a customer or a project because it violates one of these values.
4. **Differentiation**: Explain how these values are different from the [COMMON_INDUSTRY_CLICHES].

### Output Constraints:
- Use a Bold, Principles-First, and Authentic tone.
- Format the values in a styled list.
- Conclude with a "Cultural Litmus Test"—a single question the team can ask during any meeting to see if they are living the values.
```

## 🧩 Variables to Fill Out

- `[ORGANIZATION/PROJECT]` - e.g., A bootstrapped software company, a local non-profit.
- `[INSERT_WHAT_YOU_DO]` - Describe the core function and "Why."
- `[VALUE_NAME]` - e.g., Radical Transparency, Speed as a Feature, Extreme Ownership.
- `[THE_TEAM]` - Internal culture impacts.
- `[THE_CUSTOMER]` - Brand perception impacts.
- `[COMMON_INDUSTRY_CLICHES]` - e.g., "Customer First," "Quality Excellence."

## 💡 Pro-Tips

- **The Trade-Off Rule:** A value isn't a value unless it costs you something. If you value "Speed," you must be willing to sacrifice "Perfect." Ask the AI to "Identify the trade-offs for every value."
- **Use "Vivid" Language:** Instead of "Communication," use "Over-Communicate by Default." Instead of "Quality," use "Details are or Done."
- **Focus on the "Edge Cases":** Values are most useful when things are going wrong. Ask the AI: "How do these values guide us during a PR crisis or a system outage?"
