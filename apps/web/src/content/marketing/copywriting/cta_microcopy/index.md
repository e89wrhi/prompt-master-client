# Template: High-Conversion CTA Microcopy

Use this template to optimize the "last mile" of your conversion funnel: the buttons, links, and "Click Triggers" that drive the final action. It is designed to reduce user anxiety and increase motivation by transforming generic commands (like "Submit") into benefit-led invitations.

## 📋 Prompt Template

```text
Act as a Senior Conversion Strategist and Microcopy Specialist. Generate 10 variations of Call-to-Action (CTA) microcopy for [THE_BUTTON_OBJECTIVE].

### Context:
- **The Page**: [DESCRIBE_THE_PAGE] (e.g., Pricing, Newsletter Popup, Cart Checkout).
- **The User Anxiety**: [WHAT_MIGHT_STOP_THEM_FROM_CLICKING] (e.g., "Will I be charged?", "Is this a long form?").
- **The Core Benefit**: [THE_IMMEDIATE_VALUE_OF_CLICKING].

### Writing Requirements:
1. **The "Benefit-Led" Style**: Write 3 CTAs that start with a verb and lead with the value (e.g., "Get My Free Guide").
2. **The "Anxiety-Reducer" Style**: Write 3 CTAs that include a "Click Trigger" beneath or within the button (e.g., "No Credit Card Required").
3. **The "First-Person" Style**: Write 2 CTAs from the perspective of the user (e.g., "Start My Trial").
4. **The "Curiosity" Style**: Write 2 CTAs that hint at what's inside without giving it all away (e.g., "See the Results").
5. **The Feedback Loop**: Suggest the most effective [BUTTON_COLOR] and [HOVER_EFFECT] to pair with this copy.

### Output Constraints:
- Keep the main button copy under [WORD_LIMIT] words.
- Provide a brief 1-sentence "Psychological Rationale" for why each style works.
- Use a Results-Oriented and Persuasive tone.
```

## 🧩 Variables to Fill Out

- `[THE_BUTTON_OBJECTIVE]` - e.g., Newsletter Signup, Add to Cart, Book a Demo.
- `[DESCRIBE_THE_PAGE]` - Where is the button located?
- `[WHAT_MIGHT_STOP_THEM_FROM_CLICKING]` - The mental barrier.
- `[THE_IMMEDIATE_VALUE_OF_CLICKING]` - What happens in the next 10 seconds?
- `[BUTTON_COLOR/HOVER_EFFECT]` - Visual suggestions.
- `[WORD_LIMIT]` - Usually 2-5 words for a button.

## 💡 Pro-Tips

- **Active Over Passive:** Replace "Subscribe" with "Join 10,000+ Readers." Replace "Send" with "Get a Quote."
- **The Click Trigger:** Sometimes the text *around* the button is more important than the text *on* it. Ask the AI to "Design the micro-text below the button to handle the #1 objection."
- **A/B Test the Perspective:** Test "Start **Your** Free Trial" vs "Start **My** Free Trial." One focuses on ownership (My), the other on the provider (Your). Ask the AI to "Write both 'My' and 'Your' variations."
