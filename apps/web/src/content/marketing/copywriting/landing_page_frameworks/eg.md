# Examples: Landing Page Frameworks

## 1. PAS Framework (Product: Debt Consolidation)

```text
Act as a Senior Conversion Copywriter. Structure a Landing Page using the PAS (Problem-Agitate-Solution) framework for "ClearPath Debt Solutions."

### Page Goal:
Get users to sign up for a free 15-minute consultation.

### Structure Requirements:
1. **Hero**: Headline: "Stop Living Paycheck to Paycheck."
2. **Problem**: The stress of 20%+ interest rates.
3. **Agitation**: The cost of ignoring it (Bad credit, mounting debt).
4. **Solution**: Our consolidated low-interest loan.
5. **Risk Reversal**: "No fee unless we save you money."

### Output Constraints:
- Full section-by-section copy.
- Empathetic and Authoritative tone.
```

## 2. BAB Framework (Product: Luxury Eco-Resort)

```text
Act as a Senior Conversion Copywriter. Structure a Landing Page using the BAB (Before-After-Bridge) framework for "Aura Retreats."

### Page Goal:
Get users to book a 5-night "Digital Detox" stay.

### Structure Requirements:
- **Before**: The burnout and noise of city life.
- **After**: Waking up to the sound of the ocean in total silence.
- **Bridge**: Our ultra-private, Wi-Fi-free luxury eco-villas.

### Output Constraints:
- Focus on sensory imagery.
- Compassionate tone.
```
