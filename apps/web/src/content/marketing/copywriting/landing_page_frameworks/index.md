# Template: Strategic Landing Page Frameworks

Use this template to architect a high-converting landing page from the ground up. It covers the structural "Skeleton" of the page using proven copywriting formulas like PAS (Problem-Agitate-Solution) or AIDA (Attention-Interest-Desire-Action) to ensure a logical flow that leads the user toward a single conversion goal.

## 📋 Prompt Template

```text
Act as a Senior Conversion Copywriter. Structure a full Landing Page using the [SPECIFIC_FRAMEWORK] for [PRODUCT/OFFER].

### Page Goal:
[INSERT_CONVERSION_GOAL_AND_AUDIENCE]

### Structure Requirements:
1. **The Hero Section (Attention)**: 
   - **Main Headline**: A powerful [BENEFIT_DRIVEN] hook.
   - **Sub-headline**: Explaining the "How" and reducing friction.
   - **Primary CTA**: [CTA_TEXT].
2. **The Problem/Agitation Section**: Deeply empathize with the [USER_PAIN_POINT]. Describe the "Cost of Inaction."
3. **The Solution (The Big Reveal)**: Introduce [PRODUCT/OFFER] as the only logical fix. List the top 3 "Benefit-to-Feature" mappings.
4. **Social Proof & Authority**: Propose where to place [TESTIMONIALS], [CLIENT_LOGOS], or [DATA_POINTS].
5. **The Risk Reversal (The Guarantee)**: Craft a 2-sentence guarantee to remove final [BUYING_ANXIETY].
6. **The Final Call to Action (The Close)**: A high-stakes closing argument and a repeat of the primary CTA.

### Output Constraints:
- Provide the copy for each section header and body text.
- Use a [TONE] tone.
- Include "Design Notes" for each section describing the visual layout (e.g., "Use high-contrast background here").
```

## 🧩 Variables to Fill Out

- `[SPECIFIC_FRAMEWORK]` - e.g., PAS (Problem-Agitate-Solution), AIDA, BAB (Before-After-Bridge).
- `[PRODUCT/OFFER]` - What are you selling?
- `[INSERT_CONVERSION_GOAL]` - e.g., $97 Online Course, Free Webinar Signup, B2B Software Demo.
- `[USER_PAIN_POINT]` - The specific struggle the audience is facing.
- `[TONE]` - e.g., Urgent/Direct, Compassionate/Empathetic, Authoritative/Exclusive.

## 💡 Pro-Tips

- **PAS is Best for Pain:** If your product solves a painful problem (e.g., "My back hurts," "I'm losing money"), use PAS. It forces the user to confront the problem before the solution.
- **BAB is Best for Dreamers:** If your product is aspirational (e.g., "Join the 1%", "Look 10 years younger"), use BAB to paint the "After" picture immediately.
- **Kill the "We" Language:** Audit the page for "We." Replace it with "You." (e.g., instead of "We provide...", use "You get...").
