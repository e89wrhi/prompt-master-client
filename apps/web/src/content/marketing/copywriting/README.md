# ✍️ Copywriting Prompts

Templates that turn the AI into a direct-response master for landing pages and sales assets.

## Available Categories

- [**CTA Microcopy**](./cta_microcopy/index.md): Button text and helper text that reduces friction.
- [**Landing Page Frameworks**](./landing_page_frameworks/index.md): Structuring pages using PAS, AIDA, and StoryBrand frameworks.
- [**Product Descriptions**](./product_descriptions/index.md): Benefit-driven eCommerce product descriptions.
- [**Sales Letter Drafts**](./sales_letter_drafts/index.md): Long-form, highly emotional direct response letters.
