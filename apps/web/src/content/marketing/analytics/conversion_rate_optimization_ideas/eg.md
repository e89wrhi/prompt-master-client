# Examples: Conversion Rate Optimization (CRO)

## 1. SaaS Pricing Page Optimization

```text
Act as a Senior CRO Specialist. Conduct a heuristic analysis for a SaaS Pricing Page.

### Page Context:
- Product: B2B Project Management Tool.
- Current CR: 2.5% from Page Visit to Trial Signup.
- Description: 3 tiers (Basic, Pro, Enterprise).

### Analysis Requirements:
1. **Friction Points**: Long complex feature comparison table.
2. **Value Prop**: Headline is "Choose your plan."
3. **Hypotheses**:
   - If we add "Most Popular" badge to Pro, then conversion will increase because of social proof.
   - If we reduce the number of form fields from 10 to 3, then signups will increase because of reduced friction.

### Output Constraints:
- Hypotheses in a table.
- Analytical and Bold tone.
```

## 2. eCommerce Product Page (High-End Skincare)

```text
Act as a Senior CRO Specialist. Conduct a heuristic analysis for a $120 Face Cream product page.

### Page Context:
- Audience: Women 40+, high income.
- Goal: Add to Cart.
- Pain Point: "Is it worth the price?"

### Analysis Requirements:
- Identify 3 friction points.
- Propose 2 ways to implement Social Proof (e.g., Clinical trial results).
- Mobile-specific fixes.

### Output Constraints:
- Objective and Professional tone.
```
