# Examples: High-CTR Meta Descriptions

## 1. High-End Ergonomic Desk Chair (Product Page)

```text
Act as a Search Marketing Specialist. Generate 5 variations of a Meta Description for "The Zenith Pro Chair."

### Page Context:
- **Target Keyword**: Ergonomic Desk Chair.
- **Audience**: Remote workers.
- **Benefit**: 10-year warranty, free shipping.

### Writing Requirements:
- Character limit: 145-155.
- Keyword at the beginning.
- CTA: "Shop now and save."

### Output Constraints:
- Character count included for each.
- Persuasive tone.
```

## 2. Informational Guide: "How to Bake Sourdough"

```text
Act as an SEO Specialist. Generate 5 variations for an ultimate sourdough guide.

### Page Context:
- **Target Keyword**: How to bake sourdough.
- **Benefit**: Beginner-friendly, 5 simple steps.

### Writing Requirements:
- Focus on "Learn the secret to perfect crust."
- CTA: "Start your starter today."

### Output Constraints:
- Direct and Helpful tone.
```
