# Template: High-CTR Meta Description Generation

Use this template to generate precise, action-oriented meta descriptions that convince searchers to click on your link. It is designed to strictly adhere to the 155-160 character limit while incorporating primary keywords, unique value propositions, and direct calls to action (CTAs).

## 📋 Prompt Template

```text
Act as a Senior Search Marketing Specialist. Generate 5 variations of an SEO Meta Description for the following [PAGE_CONTENT].

### Page Context:
- **Target Keyword**: [PRIMARY_KEYWORD]
- **Target Audience**: [DESCRIBE_AUDIENCE]
- **Primary Benefit/Offer**: [e.g., Free Shipping, Save 20%, Learn for Free].

### Writing Requirements:
1. **The Character Limit**: Every variation MUST be between **[MIN_CHARS] and [MAX_CHARS] characters** (including spaces). This ensures it won't be truncated by Google.
2. **The Keyword Placement**: Ensure the [PRIMARY_KEYWORD] appears as close to the beginning of the description as possible.
3. **The "Benefit-Led" CTA**: Include a strong, verb-based call to action at the end (e.g., "Shop now," "Read the guide," "Save today").
4. **Unique Selling Point (USP)**: Mention one thing that differentiates this page from the other 9 results on the page.
5. **No "Dry" Summaries**: Avoid just restating the title; focus on the *reason* why the user should click.

### Output Constraints:
- Provide a character count next to each of the 5 variations.
- Use a Persuasive and Direct tone.
- Ensure the description sounds natural and is not just "keyword-stuffed."
```

## 🧩 Variables to Fill Out

- `[PAGE_CONTENT]` - Describe the page or paste the H1 and a summary.
- `[PRIMARY_KEYWORD]` - The term you want to rank for.
- `[MIN_CHARS]` - Usually 145.
- `[MAX_CHARS]` - Usually 155 (Google shows about 160 total).

## 💡 Pro-Tips

- **Google Rewrites Metas:** About 70% of the time, Google will rewrite your meta if it doesn't think it matches the user's specific query. Use secondary keywords to increase the chance Google uses your custom draft.
- **Numbers Stand Out:** "Used by 10,000+ people" or "10-minute guide" provides social proof and clarity, which increases Click-Through Rate (CTR).
- **Match the Search Intent:** If the user is searching for a tutorial, don't write a salesy meta. Use "Learn how to..." or "Step-by-step guide to..."
