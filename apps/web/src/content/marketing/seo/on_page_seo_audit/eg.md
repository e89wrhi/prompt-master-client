# Examples: Technical On-Page SEO Audit

## 1. Landing Page for "Personal Injury Lawyer NYC"

```text
Act as a Technical SEO Specialist. Performance an On-Page SEO Audit for: "Personal Injury Lawyer NYC."

### Audit Scope:
[User pastes page content: "We are a law firm. We help people. Our office is in New York..."]

### Audit Requirements:
1. **Keyword-to-Content**: Is "Personal Injury Lawyer NYC" in the H1?
2. **Title & Meta**: Draft high-CTR versions involving "Free Consultation."
3. **Internal Link Audit**: Link to "Car Accident Law" and "Medical Malpractice."
4. **Image & Alt-Text**: Alt-text for the "Office Logo" and "Attorney Bio."

### Output Constraints:
- Categorized "Audit Scorecard" format.
- Priority 1-2-3 action list.
```

## 2. Blog Post: "Top 10 Gluten-Free Recipes"

```text
Act as a Technical SEO Specialist. Audit my latest blog post.

### Audit Requirements:
- Check if keywords like "Gluten-Free Recipes" are in the H2s.
- Suggest internal links to my "Favorite GF Flour" article.

### Output Constraints:

```
