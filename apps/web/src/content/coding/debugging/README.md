# 🐛 Debugging Prompt Templates

Reusable prompt templates explicitly designed for diagnosing bugs, finding leaks, and resolving environmental hell.

## Available Templates

- [**Dependency Hell**](./dependency_hell/index.md): Resolve build, config, and package version conflicts.
- [**Memory & Performance Hunt**](./memory_leak_hunting/index.md): Isolate RAM leaks and CPU bottlenecks.
- [**Network & CORS Issues**](./network_cors_issues/index.md): Fix proxy, header, and preflight browser errors.
- [**Race Condition & Concurrency**](./race_condition_concurrency/index.md): Debug async timing, deadlocks, and thread safety.
- [**Stack Trace Analysis**](./stack_trace_analysis/index.md): Decode cryptic errors and get root-cause fixes.

To use these, copy the contents of the `Prompt Template` section into your LLM of choice.
