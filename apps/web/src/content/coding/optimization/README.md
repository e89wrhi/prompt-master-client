# 🚀 Optimization Prompt Templates

Scaffolded directories for using AI to find bottlenecks, refactor O(N^2) code, and implement caching.
*(Note: These templates are currently blank placeholders waiting to be filled).*

## Available Categories

- [**Algorithm Refactoring**](./algorithm_refactoring/index.md): Reducing Big-O complexity for core functions.
- [**Bundle Size Reduction**](./bundle_size_reduction/index.md): Webpack/Vite analyzer optimization, lazy loading.
- [**Caching Strategies**](./caching_strategies/index.md): Redis implementation, CDN invalidations, in-memory caching.
- [**Database Query Tuning**](./database_query_tuning/index.md): Using EXPLAIN plans to add indexes and optimize ORMs.
- [**Memory Profiling**](./memory_profiling/index.md): Fixing specific RAM spikes (distinct from active leak debugging).
