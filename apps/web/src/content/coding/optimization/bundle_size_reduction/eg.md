# Examples: Intensive Bundle Optimization

## 1. Next.js App Audit (Lighthouse 45 -> 90)

```text
Act as a Performance Engineer. My Next.js app bundle is 2MB. Help me reduce it.

### Context:
- **Major Libraries**: Moment.js, lodash, heavy UI icons.
- **Goal**: Hit a Lighthouse score of 90+.

### Requirements:
1. Provide alternatives for Moment.js and Lodash.
2. Show how to dynamically import the "Heavy Chart" component.
3. Suggest a Vite/Next.js config to analyze the bundle.

### Output Constraints:
- Performance-Focused and Decisive tone.
- Payload Forecast included.
```
