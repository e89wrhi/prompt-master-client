# ⚡ Code Generation Prompt Templates

Scaffolded directories for getting the AI to write huge chunks of boilerplate, complex RegEx, or data mocks rapidly.
*(Note: These templates are currently blank placeholders waiting to be filled).*

## Available Categories

- [**Boilerplate Scaffolding**](./boilerplate_scaffolding/index.md): Generating initial project structures or repetitive configs.
- [**Component Generator**](./component_generator/index.md): React/Vue/Svelte UI component generation with styling.
- [**Data Mocking**](./data_mocking/index.md): Generating fake JSON seed data or mock database rows.
- [**Regex Builder**](./regex_builder/index.md): Constructing complex, validated Regular Expressions.
- [**Script Automation**](./script_automation/index.md): Bash, Python, or PowerShell scripts for automating dev tasks.
