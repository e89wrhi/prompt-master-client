# Examples: High-Aesthetic UI Generation

## 1. Glassmorphic Pricing Card (React + Tailwind)

```text
Act as a UI Engineer. Forge a "PricingCard" component using React and Tailwind CSS.

### Context:
- **Design**: Premium glassmorphism (translucent background, blur effect, subtle white border).
- **Features**: A "Recommended" badge, a list of features with checkmarks, and a call-to-action button.

### Requirements:
1. Provide the TypeScript interface for `title`, `price`, `features`, and `isRecommended`.
2. Use smooth hover transitions and custom HSL colors.
3. Include ARIA labels for the list and button.

### Output Constraints:
- Aesthetic and Code-Quality-Focused tone.
- Conclude with Storybook Draft.
```
