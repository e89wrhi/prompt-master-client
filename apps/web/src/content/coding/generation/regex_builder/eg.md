# Examples: High-Precision Regex Architect

## 1. Complex Email Validation (RFC 5322)

```text
Act as a Regular Expression Expert. Create a regex for validating emails that follow RFC 5322 standards.

### Context:
- **Language**: JavaScript.
- **Goal**: Validate complex subdomains and specific TLDs.

### Requirements:
1. Provide the pattern.
2. Break down the logic of the local part vs the domain part.
3. Provide 3 valid and 3 invalid test cases (e.g., `user+tag@domain.co.uk` vs `user@@domain.com`).

### Output Constraints:
- Precise and Technical tone.
```

## 2. Parsing Apache Log Dates

```text
Act as a String Pattern Expert. Create a regex to extract the Timestamp, IP, and Request Path from an Apache log line.

### Requirement:
- Create specific capture groups for each item.
- Explain the logic for the date-time brackets `[dd/MMM/yyyy:HH:mm:ss Z]`.

### Output Constraints:
- Include Capture Group Map.
```
