# Examples: Professional Boilerplate & Scaffolding

## 1. Next.js SaaS Monorepo (T3 Stack Style)

```text
Act as a Full-Stack Architect. Scaffold a SaaS application for a "Project Management" tool.

### Context:
- **Stack**: Next.js 14 (App Router), TypeScript, Tailwind, Prisma, Clerk (Auth).
- **Structure**: Feature-based modules in `src/modules`.

### Requirements:
1. Provide the folder map.
2. Provide the `tailwind.config.ts` and `next.config.js`.
3. Provide a list of essential dependencies for my `package.json`.

### Output Constraints:
- Scalable and Formal tone.
- Include "The Maintainer's Rulebook."
```

## 2. Rust CLI Boilerplate (Clap + Serde)

```text
Act as a Project Lead. Scaffold a Rust CLI tool for "Log Parsing."

### Requirement:
- Code for `Cargo.toml`.
- Folder structure for a modular CLI.
- Entry point `main.rs` with basic argument parsing using `Clap`.

### Output Constraints:
- Architecturally-Sound tone.
```
