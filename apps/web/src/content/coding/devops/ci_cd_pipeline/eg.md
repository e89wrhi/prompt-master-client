# Examples: Strategic CI/CD Automation

## 1. GitHub Action for Next.js (Lint, Test, Vercel Deploy)

```text
Act as a CI/CD Specialist. Design a GitHub Action workflow for my Next.js site.

### Context:
- **Major Jobs**: Lint, Unit Tests (Jest), Build, and Deploy.
- **Trigger**: Push to `main` for Production, and PRs for Staging.
- **Optimization**: Use `actions/cache` for `node_modules` and high-speed build.

### Output Constraints:
- YAML format.
- Include a "Manual Approval" step for Prod.
```
