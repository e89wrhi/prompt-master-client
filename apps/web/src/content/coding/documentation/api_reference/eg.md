# Examples: Professional API Reference

## 1. Payments API Reference (/v1/charges)

```text
Act as a Technical Writer. Draft the reference for the `/charges` resource of a Payments API.

### Context:
- **Actions**: POST (Create Charge), GET (Retrieve Charge), GET (List Charges).
- **Target**: External developers integrating our payment gateway.

### Requirements:
1. Provide the Endpoint Overview table.
2. Show a 201 Created response for a successful charge.
3. Show a 402 Payment Required error message structure.

### Output Constraints:
- Formal and Industry-Standard tone.
- Include a cURL example.
```
