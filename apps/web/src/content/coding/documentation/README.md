# 📚 Documentation Prompt Templates

Scaffolded directories for generating high-quality documentation, comments, and architecture records using AI.
*(Note: These templates are currently blank placeholders waiting to be filled).*

## Available Categories

- [**API Reference**](./api_reference/index.md): OpenAPI/Swagger generation, JSDoc decorators.
- [**Architecture ADR**](./architecture_adr/index.md): Architecture Decision Records and system design outlines.
- [**Code Comments**](./code_comments/index.md): Inline and block-level explanatory comments for complex logic.
- [**README Generator**](./readme_generator/index.md): Project-level descriptions, setup instructions, and badges.
- [**Release Notes**](./release_notes/index.md): Digesting git commits into user-facing changelogs.
